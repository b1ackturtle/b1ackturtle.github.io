Booker - 書籍管理 (Booker - Book Manager)

This app is a Simple Book Manager that helps you organize and catalog your book collection.

The main features are the following.

  • Easy to manage your books(title, author, publisher, published date, thumbnail...)
  • Automatic download book data from GoogleBooks using ISBN code
  • Quickly add books by scanning barcode
  • Search registered books by title, author, and tag
